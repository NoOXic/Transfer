KvK Transfer Analysis — Alliance TFR | Server 517 Migration Research

What This Tool Does

This dashboard helps Alliance TFR (Server 517) evaluate 59 potential target servers for a KvK migration. Each server is rated against your current baseline using multiple competitive metrics. The goal: find a server that offers the right challenge — strong enough to grow, not so strong that TFR gets crushed on arrival.

Key Metrics

KO Rating
Bradley-Terry model score. Average = 1.0. Your server 517 = 0.770 (below average).
Prep Record ★
Most predictive metric. Strong prep win rate reliably forecasts future KvK performance.
KS Atlas Tier
Independent tier rating (S→D). Cross-checks the KO rating.
Score (0–100)
Composite score: 35% KO + 35% Prep + 15% Guilds + 15% MT Players. Easy summary for officers.
Active Guilds
Guilds with 14B+ power. Indicates server health and how much competition TFR will face.
MT Players 2000+
High-level active players. Low count may mean the server is dying.

Verdict System

Recommended
Passes all strict criteria. Strong migration target.

Good but doesn't clear every threshold. Worth researching.
Neutral
Lateral move — similar strength to 517.
Not Rec.

Tips for TFR

  • Prioritise servers where Prep Win% ≥ 70% — strongest predictor of what you'll face.
